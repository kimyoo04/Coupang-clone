# Coupang-clone (쿠팡 클론코딩)

## 프로젝트 소개

- 쿠팡 사이트를 클론코딩한 프로젝트입니다.
- 프론트엔드와 백엔드를 모두 구현하였습니다.
- 프론트엔드는 Next.js를 사용하였고, 백엔드는 Nest.js를 사용하였습니다.
